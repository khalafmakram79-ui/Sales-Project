# Sales Dashboard

### وصف المشروع
لوحة تحكم تفاعلية لتحليل أداء المبيعات. 
تم جمع البيانات من خلال Google Forms و تخزينها على Google Sheets 
وعمل الـ Dashboard على Looker Studio.

الهدف: متابعة الإيرادات، أداء فريق المبيعات، وتوزيع المبيعات حسب العلامات التجارية والشركات بشكل لحظي.

### أهم المؤشرات KPIs
- **إجمالي الإيرادات**: 1.2M جنيه
- **عدد الطلبات**: 544 طلب

### أهم التحليلات الموجودة في الـ Dashboard
1.  **أداء المندوبين**: 
    مخطط أعمدة بيقارن مبيعات كل مندوب. 
    أعلى مندوبين: عمر شريف و أحمد سامي بإجمالي 300K لكل منهم.
    
2.  **توزيع المبيعات حسب العلامة التجارية**:
    مخطط دائري بيوضح نسبة كل براند.
    - Dell: 45.8%
    - Samsung: 37.3%
    - HP: 9.4%
    - Logitech: 7.4%

3.  **أهم العملاء والشركات**:
    مخطط أفقي بيعرض أكبر 10 شركات من حيث حجم المشتريات.

4.  **توزيع المبيعات حسب الفئة**:
    مخط TreeMap بيوضح نسب المبيعات بين الفئات: الإلكترونية، الدفترية، ...

### الأدوات المستخدمة
- **Google Forms**: لجمع بيانات المبيعات من المندوبين
- **Google Sheets**: لتخزين وتنظيف البيانات
- **Looker Studio**: لعمل الـ Dashboard التفاعلي وعرض النتائج
